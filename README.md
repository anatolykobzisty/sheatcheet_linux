#### Работа с терминалом

[видео](https://youtu.be/jP2dsNqXeYU)

| команды           | описание                                           |
| ----------------- | -------------------------------------------------- |
| uname -a          | информация об OC                                   |
| lscpu             | данные процессора                                  |
| man команда       | помощь по названию команды                         |
| info команда      | тоже помощь по названию команды                    |
| uptime            | время с последнего включения компьютера            |
| whatis команда    | показывает что делает команда                      |
| whereis команда   | показывает где файл                                |
| locate            | показывает где файл                                |
| echo text text    | выводит текст                                      |
| echo "text text"  | выводит текст, как одно слово                      |
| ls                | показать что в этой директории                     |
| ls -l             | показать что в этой директории подробно            |
| ls -al            | показать что в этой директории со скрытыми файлами |
| ls –la –R /       | показать все на компьютере                         |
| Сtrl+Alt+F2 слева | вывести на весь экран окно терминала               |
| Сtrl+Alt+F1 слева | свернуть окно терминала                            |
| Ctrl+Z            | отправить процесс на background                    |
| Ctrl+C            | прекратить процесс вообще                          |

#### Работа с файлами

[видео](https://youtu.be/GZkeTXzMGh0)

| команды                   | описание                                                                |
| ------------------------- | ----------------------------------------------------------------------- |
| touch file.txt            | создать файл или обновить время создания файла                          |
| cat file.txt              | выводит все что есть в файле на экран                                   |
| less file.txt             | выводит все что есть в файле на экран полностью                         |

| mv file.txt NewFile.txt   | переименовать файл                                                      |
| mv file.txt .NewFile.txt  | скрыть файл                                                             |
| mv file.txt ~/Desktop/    | перенести файл                                                          |
| rm file.txt               | стереть файл                                                            |
| rm \*                     | стереть все файлы в директории                                          |

#### Работа с текстовыми данными

[видео](https://youtu.be/S4TjfU_ivpY?t=2)

| команды                          | описание                                                                  |
| -------------------------------- | ------------------------------------------------------------------------- |
| find /home -name "\*.txt"        | поиск всех файлов .txt                                                    |
| find /home -name "\file\*.txt"   | поиск всех файлов .txt названия которых начинается с file                 |
| wc file.txt                      | посмотреть сколько строчек,слов,символов в файле                          |
| wc -l                            | посмотреть сколько строчек в файле                                        |
| wc -w                            | посмотреть сколько слов в файле                                           |
| sort file.txt                    | сортирует данные в файле и выводит на экран                               |
| sort file.txt -n                 | сортирует числа в файле и выводит на экран                                |
| cut -d ">" -f 3 file.txt         | выводит текст в 3 столбце с разделителем ">" в один столбец               |
| cut -d ">" -f 3 file.txt \| sort | выводит текст в 3 столбце с разделителем ">" в один столбец c сортировкой |

#### Команда grep и регулярные выражения

[видео](https://youtu.be/Y1fjIG7S_F0)

| команды                               | описание                                                                            |
| ------------------------------------- | ----------------------------------------------------------------------------------- |
| greb word ./\*                        | поиск слова word (нижний регистр) в текущей директории и вывод строк с этим словом  |
| greb Word ./\*                        | поиск слова word (верхний регистр) в текущей директории и вывод строк с этим словом |
| greb -i word ./\*                     | поиск слова word (любой регистр) в текущей директории и вывод строк с этим словом   |
| greb word /dir/file.txt               | поиск слова в определенной директории                                               |
| greb [A-Z]\* file.txt                 | поиск любого слова из больших букв                                                  |
| greb [0-9]\* file.txt                 | поиск сколько угодно подряд стоящих цифр                                            |
| greb [A-Za-z]_@[A-Za-z]_.com file.txt | поиск емайлов с окончанием .com                                                     |

#### Перенаправление вывода и ввода

[видео](https://youtu.be/tMofCc4RX1k)

| команды                                    | описание                                                    |
| ------------------------------------------ | ----------------------------------------------------------- |
| sort file.txt > file_sorted.txt            | сохранение отсортированного файла                           |
| sort file2.txt >> file1.txt                | добавить данные из файла file2 в file1                      |
| sort file.txt >> file.txt                  | сортировка и сохранение этого же файла                      |
| greb word /dir/\* 2> errors.txt            | перенаправление ошибок в errors.txt                         |
| greb word /dir/\* 2> /dev/null             | перенаправление ошибок в никуда                             |
| greb word /dir/\* > file.txt 2> errors.txt | перенаправление успешных ответов и ошибок в отдельные файлы |
| greb word /dir/\* > &> file.txt            | перенаправление успешных ответов и ошибок в один файл       |
